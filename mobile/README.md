# ForAll Mobile App

Мобильное приложение ForAll на React Native с Expo.

## Установка и запуск

1. Установите зависимости:
```bash
npm install
```

2. Запустите приложение:
```bash
npm start
```

3. Отсканируйте QR-код в приложении Expo Go на вашем телефоне или запустите на эмуляторе:
- Android: `npm run android`
- iOS: `npm run ios`

## Настройка API

Откройте `src/config/api.js` и измените `API_BASE_URL` на адрес вашего backend сервера.

Для локальной разработки используйте ваш локальный IP адрес вместо `192.168.1.100`.
